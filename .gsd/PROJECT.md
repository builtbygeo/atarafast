# Project GSD

**Name:** Atara (Open Source Maturity)
**Vision:** A minimalist, functional intermittent fasting tracker. Currently a fully open-source PWA with democratic access to AI analysis and advanced stats.
**Status:** Stabilization and open-source transition complete. Moving to M002: Democratizing premium features and adding advanced user tracking (weight).

## Overview

Atara is transitioning from a premium model to a community-driven open-source product. The immediate focus is removing paywalls for AI and dashboard features, implementing usage-based limits instead, and adding personalized health tracking (weight) without compromising the compact, non-scrolling UI.

## Milestones

- **M001:** Stabilization & Open Source Preparation
  Stabilized fasting calculations, fixed notification UX, and introduced base feature gating.
- **M002:** Democratization & Personalization
  Unlocking premium features for all users, implementing AI quotas, and adding weight tracking.
