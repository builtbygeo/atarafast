# AtaraFast App – Open Source Core

**Master Your Metabolism**  
AI-powered fasting timer, progress tracker and electrolyte insights.

This is the **open-source core** (AGPL-3.0) of the AtaraFast app (hosted at https://app.atarafast.com).

The landing page (https://atarafast.com) is a separate closed-source Vercel project.

## ✨ What's included (open core)
- Fasting timer & streak counter
- Weekly activity & progress graphs
- Basic electrolyte & minerals tracker
- Calendar of past & future fasts
- Responsive PWA (works offline)
- Clean, modern UI

## 🚀 Pro features (hosted version only)
- Advanced AI metabolism insights
- Custom protocol builder
- Premium electrolyte recommendations
- Stripe subscription & account system
- Priority updates & support

## How to self-host (open core)
1. Clone the repo
2. `npm install`
3. Add your own `.env` file with necessary API keys for basic features
4. `npm run dev`

**Important note:** Advanced AI features, subscription logic, and premium protocols are only available in the official hosted version at https://app.atarafast.com.
